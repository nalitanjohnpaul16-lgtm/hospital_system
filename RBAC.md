Role Base Access-Control
---------------------------------------------------------------------------------------------------------------------------------------------

---------------------------------------------------------------------------------------------------------------------------------------------
Admin
- FULL access but with verifications
- can ban accounts
- can access all the access as auditor such as view archive, start new session, and restore old audit logs.


---------------------------------------------------------------------------------------------------------------------------------------------
Doctor

- Decrypt/encrypt with verification to Doctors Records, Appointment records, Patients, Diagnosis, and Prescriptions.
- Edit with Verification to Patients, Diagnosis, Prescriptions, Incident Reports, Doctor Module, Appointments  
- READ/VIEW/Export CSV to other modules. 

---------------------------------------------------------------------------------------------------------------------------------------------
Nurse

- Decrypt/encrypt with Verification to Doctors Records, Appointment records, Patients, Diagnosis, and Prescriptions.
- Edit with Verification method to Patients, Incident Reports, Medical Store
- READ/VIEW/Export CSV to other modules. 

     
---------------------------------------------------------------------------------------------------------------------------------------------
Auditor

- Edit audit logs such as view archive, start new session, and restore old audit logs.
- READ/VIEW/Export CSV to other modules.
---------------------------------------------------------------------------------------------------------------------------------------------
IT Security

- Decrypt/encrypt with Verification to Doctors Records, Appointment records, Patients, Diagnosis, and Prescriptions.
- Edit with Verification method to Assets, Threats, BIA, Incident Reports
- READ/VIEW/Export CSV to other modules.
- can access all the access as auditor such as view archive, start new session, and restore old audit logs.

---------------------------------------------------------------------------------------------------------------------------------------------
Asset Manager

- Decrypt/encrypt with Verification to Doctors Records, Appointment records, Patients, Diagnosis, and Prescriptions.
- Edit with Assets Inventory Records
- READ/VIEW/Export CSV to other modules.
---------------------------------------------------------------------------------------------------------------------------------------------
Inventory Staff

- Decrypt/encrypt with Verification to Doctors Records, Appointment records, Patients, Diagnosis, and Prescriptions.
- Edit with Medical Store Records and Incident Reports
- READ/VIEW/Export CSV to other modules.


---------------------------------------------------------------------------------------------------------------------------------------------
Patients and Other (e.g. Parents, Guardian etc.)

- Decrypt/encrypt with Verification to Doctors Records, Appointment records, Patients, Diagnosis, and Prescriptions.
- READ/VIEW/Export CSV to modules.

---------------------------------------------------------------------------------------------------------------------------------------------

ALL ROLES:

- can remove/save/delete account in the system.
- can be ban by Admin (BUT ADMIN CAN'T BAN ITSELF) 
- audit the actions
- can access search bar for each records of all modules
- add the information's of users to DASHBOARD module
